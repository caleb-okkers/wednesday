There are different programming languages that involve web development

HTML

HyperText Markup Langauge is responsible for providing structure, content and semantics to the web page

CSS

Cascading Style Sheets is responsible for styling web pages, presentation of HTML elements and allows for separation of content and presentation for easier maintenance and updates

Javascript

Javascript is used for adding interactivity and dynamic behavior to web pages.

It also enables manipulation of HTML and CSS, allowing for changes to be made in response to user actions or events.
